## CONTENT FORMULA (what makes it rich-but-minimal, not just the look)
The Fable 5 image works because of content discipline, not decoration. Every gold image must have:
1. **ONE claim, proven** — a single headline argument. Never 5 competing ideas. (The cost image failed by stacking thesis + prices + margins + controls.)
2. **3 proofs, each a real named source with real comparison numbers** — benchmark names, competitor figures, specifics. Vague = dead. Bars carry it.
3. **A human aside on each block** — the "so what" in Dhruv's voice ("this is what agents actually do all day").
4. **ONE quotable punchline** in the callout box — the screenshot line ("that's a step, not an iteration").
5. **A hard-facts bottom strip** — the practical 3 (context / price / date / whatever the reader needs).
6. **A POV in the speech bubble** — an opinion, not neutral reporting ("this is a tier jump").
7. **Minimal prose** — numbers + bars + doodles do the talking. If a block needs a sentence, cut it to 4 words.

Test: one claim, three proofs, one punchline. If you can't say it that simply, it's not ready.

## LEAK FIX (Gemini sometimes draws the layout labels)
The `=== PANEL / CENTER / HERO ===` markers occasionally render as literal text on the image. To prevent it, add this line near the top of every prompt:
> "IMPORTANT: do NOT draw any layout words like CENTER, PANEL, HERO, BAR CHART, or any === markers as visible text. Those are instructions only."
And prefer plain sentences ("The main element is one sticky note headed ...") over `=== HEADER ===` blocks when a leak has happened before.

## RICHNESS LAYER (what makes content rich, not medium)
A bar/number alone reads flat. Add a short **grey handwritten "so-what" aside next to each bar** — the human takeaway in Dhruv's voice (Fable 5: "this is what agents actually do all day"; cost image: "most companies, unknowingly" / "ONE decision" / "pooled, not metered alone"). The asides ARE the richness. Without them the image looks medium.

## Non-negotiables (the recurring misses)
1. **Gun-metal silver etched pen** — never blue.
2. **Bars, not text stacks** — every comparison is a hand-drawn bar chart.
3. **3 panels max** — sparse, phone-readable in <10s (see RULE #0 in STYLE-LOCKED.md).
4. **Large realistic portrait**, black short-sleeve tee, never hoodie.
5. **Callout box = quote only**, never the word "callout".
6. **AI logo doodles** beside the bars = the viral hook.
